Descripción

El sistema de inventario virtual, se desarrolló como una necesidad de la Asociación de Estudiantes de la Universidad de Costa Rica, con su sede en el Pacífico Central. La misma contiene módulos de control de distintas áreas, de las cuales la asociación es encargada. Entre ellas, se puede mencionar el módulo de limpieza, alimentos y manejo de finanzas.

    Descripción

    Find Code

    Descripción

    El sistema Find Code, se realizó a partir de la necesidad que se tenía por parte del departamento financiero de la Municipalidad de Orotina, para poder ejecutar una búsqueda más rápida de los códigos únicos para cada proyecto que se ejecuta año con año en la municipalidad. Esto llega a ahorrar tiempo de búsqueda que se realizaba en un documento de Excel y el cual únicamente mostraba el código del activo, sin incluir actividad o proyecto específico. Además de que no se tomaba en cuenta el número único generado para realizar la consulta del presupuesto que se tenía otorgado a la actividad por la que se consultaba.
